Masterclass Front End Guide

Please refer to this guide when developing new features. Stick to our colors, keep our product on brand, and let's make the best site we can make!

Colors

These colors define the Masterclass brand. They're located in the variables.scss file, which you can import into any scss file you work on.

Importing

If you're working on your own scss file in the app and would like to use these colors, you'll need this at the top of your scss file.

@import "variables";

Palette

These are the brand colors we've currently defined. If they change, these will be updated automatically.

  • Primary Accents

    • $mc-color-accent-primary

    • $mc-color-accent-primary-hover

  • Secondary Accents

    • $mc-color-accent-secondary

    • $mc-color-accent-secondary-hover

    • $mc-color-text-link

    • $mc-color-text-link-hover

  • Structural

    • $mc-color-bg-fallback

    • $mc-color-neutral

  • Other

    • $mc-color-active-state

Responsive

The Masterclass core is built "mobile first". All media query overrides are based on a min-width, so the basic CSS loads first.

Breakpoints

$mc-bp-tiny:      480px;
$mc-bp-small:     768px;
$mc-bp-medium:    992px;
$mc-bp-large:     1200px;
$mc-bp-full:      1440px;
$mc-bp-uwd:       2000px;

Usage

@media (min-width: $mc-bp-full) { // SCSS here }
